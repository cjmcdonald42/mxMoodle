# Middlesex's Block for the Transportation Manager
Moodle block plugin written for Middlesex. Middlesex is an independent secondary school for boarding and day students in grades 9-12. Learn more at <https://mxschool.edu>.

Moodle is the world’s open source learning platform. Learn more at <https://moodle.org>.

## Package Description
This block provides an interface for the transportation manager to access the vacation travel transportation report defined in the local_mxschool plugin.
This block was deprecated in v3.4 and will be deleted.

## Dependencies
This plugin has the following dependencies:
- Moodle 3.7 (2019052000)
- [local_mxschool v3.1](/local/mxschool/README.md) (2019081400)
